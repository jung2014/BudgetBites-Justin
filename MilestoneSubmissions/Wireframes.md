## Notes

- All pages include a consistent navigation bar at the top
- The application uses a responsive design that adapts to different screen sizes
- Color scheme: Green/blue gradient theme with professional styling
- Interactive elements: Buttons, forms, cards, and modals are clearly indicated
- User feedback: Success/error messages are displayed as needed
- Accessibility: Forms include proper labels and validation messages

## Page Flow

1. **Unauthenticated Users:**
   - Login → Register (or vice versa)
   - After login → Dashboard

2. **Authenticated Users:**
   - Dashboard → Discover / Favorites / Settings
   - Discover → Recipe Detail → Grocery List
   - Favorites → Recipe Detail → Grocery List
   - Settings → Save preferences
   - Any page → Logout → Logout confirmation
